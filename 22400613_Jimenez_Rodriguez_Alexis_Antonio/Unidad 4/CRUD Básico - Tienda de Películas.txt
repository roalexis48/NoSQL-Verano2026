CRUD Básico - Tienda de Películas (API REST)
Flujo Cliente-Servidor solicitado.

1. Crear una película
Método HTTP: POST
URL: /movies
JSON enviado:
{
  "id":1,
  "nombre":"The Maze Runner",
  "director":"Wes Ball",
  "anio":2014,
  "duracion":1,
  "genero":"Ciencia ficción"
}
JSON recibido:
{ "code":200, "msg":"Película creada" }
2. Consultar una película por su id
Método HTTP: GET
URL: /movies/1
JSON enviado:
vacio por que es peticion 
JSON recibido:
{ "code":200, "pelicula":{...} }
3. Actualizar año, director y duración por id
Método HTTP: PUT
URL: /movies/1
JSON enviado:
{
 "director":"Joaquin Guzman",
 "anio":2021,
 "duracion":150
}
JSON recibido:
{ "code":200, "msg":"Película actualizada" }
4. Borrar una película por su id
Método HTTP: DELETE
URL: /movies/1
JSON enviado:
Sin body
JSON recibido:
{ "code":200, "msg":"Película eliminada" }
Estructura de cada película
{
  "id": Number,
  "nombre": String,
  "director": String,
  "anio": Number,
  "duracion": Number,
  "genero": String
}
